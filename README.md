# Nodejs-RESTful-API
Developed a RESTful API for a car rental service - MVC framework followed. <br/>
### Features : ### 
SignUp/Login <br/>
Add car <br/>
Delete car <br/>
View cars <br/>
Display details of a specific car <br/>
Filter based search <br/>
Book car <br/>
See booked cars <br/>
Update car details <br/>
